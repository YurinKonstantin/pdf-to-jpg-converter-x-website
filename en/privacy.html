This privacy policy governs your use of the software application PDF to JPG Converter X for Windows devices.

1. User Privacy & Data Collection

Our core principle is absolute data privacy. The Application does not collect, store, transmit, or share any personal data, user files, documents, or analytics information.

2. Local File Processing

All PDF rendering, image conversion, resizing, and printing operations are performed strictly locally on the user's device. No data is ever uploaded to external cloud servers, websites, or third-party APIs.

3. Internet Access & Telemetry

The Application works completely offline and does not require an internet connection to function. It does not contain any background tracking scripts, crash telemetry reporting, or advertising modules.

4. Contact Us

If you have any questions regarding privacy while using the Application, please contact us via our official repository channels.
